## üé® DESIGN PRINCIPLES

### Core Principles for All Images

1. **Self-Explanatory:** Images should immediately convey the service/feature/content without needing text
2. **Premium Feel:** High-quality, professional, modern aesthetic that elevates the component
3. **Content Enhancement:** Images should complement and amplify the message, not distract
4. **Brand Consistency:** Include brand colors (#3B82F6 blue, #8B5CF6 purple, #06B6D4 cyan) as accent elements
5. **Visual Hierarchy:** Important content areas should be clear, avoiding text overlay zones
6. **Mobile Responsive:** Images work well on all screen sizes with appropriate cropping

## üìù NOTES FOR DESIGNER

### Critical Considerations:

1. **Text Overlay Zones:** Many components have text overlays. Ensure important visual content is NOT in:
   - Bottom areas (where text often appears)
   - Center areas (where titles/headings appear)
   - Left/right edges (where buttons/CTAs may appear)

2. **Gradient Overlays:** Many components use gradient overlays. Design images that work well with:
   - Dark overlays (for light text)
   - Light overlays (for dark text)
   - Color overlays (brand color gradients)

3. **Hover Effects:** Some components have hover effects that reveal content. Ensure images:
   - Look good in both default and hover states
   - Support reveal animations
   - Work with opacity transitions

4. **Mobile Responsiveness:** Images will be cropped on mobile. Ensure:
   - Important content is in center areas
   - Images work well when cropped
   - Key elements remain visible on all screen sizes

5. **Brand Consistency:**
   - Use brand colors (Blue #3B82F6, Purple #8B5CF6, Cyan #06B6D4) as accent elements
   - Maintain consistent visual style across all images
   - Professional, modern aesthetic throughout

6. **Self-Explanatory:**
   - Each image should immediately convey its service/feature/content
   - No need for text to understand what the image represents
   - Clear visual communication

7. **Premium Feel:**
   - High-quality photography or illustration
   - Professional, polished appearance
   - Modern, sophisticated aesthetic
   - Elevates the component's perceived value
