# Link Innovations - Complete Website

A fully animated, interactive website built with vanilla HTML, CSS, and JavaScript using GSAP animation library.

## üéØ Features

This website integrates 6 premium GSAP components into a seamless user experience:

### 1. **FLIP Loading Animation**
- Animated preloader showing "LINK INNOVATIONS"
- Smooth transitions between different layout states
- Automatically transitions to main content after 3.5 seconds

### 2. **Hero Section with ScrollTrigger Zoom**
- Parallax zoom effect on scroll
- Pinned hero image with scale animation
- Smooth transition into services section

### 3. **Services Showcase with ScrollSmoother**
- Smooth scrolling with parallax effects
- Speed-controlled elements at different scroll rates
- Featured services:
  - IT Staffing & Recruitment
  - Cloud & Application Services
  - Data Analytics & AI

### 4. **Infinite Scrolling Service Cards**
- Horizontal snap-scrolling card carousel
- Seamless infinite loop animation
- Interactive prev/next buttons
- Scroll-based card navigation

### 5. **Team Carousel**
- Vertical 3D carousel with depth effects
- 6 team members with profiles
- Multiple navigation options:
  - Arrow buttons
  - Dot indicators
  - Keyboard arrows
  - Direct card click
- Smooth transitions with grayscale effects

### 6. **Mega Menu Navigation**
- Animated overlay menu with GSAP
- Background image transitions on hover
- SplitText character animations
- Responsive menu items:
  - Home
  - About Us
  - Services
  - Industries
  - Contact
